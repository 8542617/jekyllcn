jekyllcn
========

[http://jekyllcn.com](http://jekyllcn.com)

jekyllcn 是 [http://jekyllrb.com](http://jekyllrb.com) 的中文翻译网站

## 认领方法

1. 在想要认领的文档 ( /docs/xxx.md ) 的头信息中添加`contributor: xxx`
2. 提交一个 pull request
3. 等待 pull request 被 merge 即可开始翻译
